Portfolio

Reddit Clone

Reddit Clone

SPA front end clone of reddit.com written in React, that fetches and displays posts using the Reddit API. Posts on the main homepage and each subtopic’s homepage can be sorted by hot, new, top and rising. Posts display all relevant post information plus comments.

Strict tracking protection may need to be disabled if the site does not display correctly.

What's For Dinner

What's For Dinner

What's For Dinner uses the Spoonacular API to search for recipes using various search parameters such as prep/cooking time, macronutrient contents, cuisines, special diets, or by ingredient. Search for recipes by title, or generate random recipes.

Weather App

Weather App

Using the One Call API and Geocoding API from Open Weather, this fully responsive weather app gives the current weather forecast for any location, as well as a 48 hour and 7 day forecast. Written in HTML, CSS and JavaScript.

Tetris

Tetris

One of the most iconic games of all time, this version of Tetris is written using HTML5, CSS and JavaScript. It saves high scores using local storage, has a pause/resume feature and even plays the Tetris theme tune.

Space Invaders

Space Invaders

This version of Space Invaders is modeled on the original Japanese shooting video game released in 1978. Playable directly in the browser, using the arrow keys to move left and right, plus the spacebar to shoot.

Snake

Snake

A simple remake of the classic arcade game, this version of Snake is written using HTML5, CSS and JavaScript, and also saves high scores using local storage.
